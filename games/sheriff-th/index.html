ตั้งค่าโหมดเสริม

ผู้เล่นที่เข้าร่วม (จาก Duel)

    Host กดเริ่มได้ทันทีเมื่อผู้เล่นจากห้อง Duel อยู่ในช่วง 3-24 คน

    กระดานกลาง

    ระบบทิ้งการ์ด / จับการ์ด

    ระบบจะคำนวณแต้มการ์ดอัตโนมัติจากการ์ดในมือ และใช้ข้อมูลนี้ไปคำนวณโบนัสตอนจบเกม

    ผู้เล่น 💰 เงิน 🃏 แต้มการ์ด 🎁 โบนัส 🏆 รวม ปรับเงิน

    สรุปเกม

    ขั้นตอนการเล่น (สั้น)

    • 1️⃣ Host เลือกรอบตำรวจต่อคน (1 หรือ 2 รอบ)
    • 👥 ระบบดึงรายชื่อจาก Duel อัตโนมัติ
    • 🚨 กดเริ่มเกม ระบบสุ่มตำรวจรอบแรก + คนเข้าตลาดทันที
    • 🎴 เล่นรอบนั้น แล้วกดสุ่มรอบถัดไป
    • 🏁 ครบทุกรอบ ระบบสรุปผู้ชนะอัตโนมัติ

    ระบบโบนัส

    • 🌾 ข้าวสาร: อันดับ 1 = +15, อันดับ 2 = +10
    • 🥚 ไข่ไก่: อันดับ 1 = +12, อันดับ 2 = +8
    • 🥬 ผักสด: อันดับ 1 = +10, อันดับ 2 = +6
    • 🐟 ปลาทู: อันดับ 1 = +8, อันดับ 2 = +4
    • 🚬🍷🦐🌿 ของเถื่อน (นับรวมทุกชนิด): อันดับ 1 = +10, อันดับ 2 = +5
    • 🤝 ถ้าจำนวนการ์ดเสมอกัน ผู้เล่นทุกคนที่เสมอจะได้โบนัสอันดับนั้นเท่ากัน
    • 🧮 คะแนนรวมต่อคน = เงิน + แต้มการ์ด + โบนัส
